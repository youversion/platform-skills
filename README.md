# platform-skills
Skills for GenAI usage of YouVersion Platform APIs and SDKs.

## Available skills

- `youversion-platform-js`: JavaScript/TypeScript Node.js usage for `@youversion/platform-core`.
- `youversion-platform-react`: React SDK usage for `@youversion/platform-react-ui`, `@youversion/platform-react-hooks`, and `@youversion/platform-core`.
